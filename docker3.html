常用指令

顯示已建構的image


                    docker ps

使用Dockerfile創建image


                    docker build -t image名字:image版本 .

image版本不一定要寫，後面的 '.' 代表路徑，必須要有

輸入 docker images 查看是否創建成功

創建並啟動 container


                    docker run --name container名字 -p 8080:80 -d image名字
