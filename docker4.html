Laradock

  1. 下載laradock套件

    
                            git clone https://github.com/Laradock/laradock.git

  2. 修改laradock內的設定檔

    
                            cd laradock
                        
    
                            cp env-example .env
                        
    
                            vim .env

    app code path host 專案路徑

    php 版本，laravel 9以上需要安裝php 8.0以上版本

  3. 啟動專案需要套件 (nginx 和 mysql)

    
                            sudo docker-compose up -d nginx mysql

  4. 進入 Workspace 容器

    
                            sudo docker-compose exec workspace bash

更新php版本

  1. 在laradock 裡的env找到php version，改成要更新的版本

  2. 重新建立php-fpm 和workspace

    
                            docker-compose build php-fpm
                        
    
                            docker-compose build workspace

  3. 重新建立laradocker容器

    
                            docker-compose down
                        
    
                            docker-compose up -d nginx mysql phpmyadmin redis workspace

  4. 進入容器查看php版本

    
                            docker-compose exec workspace bash
                        
    
                            php -v
