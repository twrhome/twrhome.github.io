Docker 安裝 python 和 ELK 環境

此安裝在centos系統下

  1. 安裝docker 
    
                            curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
  2. 啟動docker 服務 
    
                            sudo systemctl start docker
  3. 安裝 docker-compose 
    
                            sudo curl -L "https://github.com/docker/compose/releases/download/1.24.1/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
                        
    
                            sudo chmod +x /usr/local/bin/docker-compose
  4. 安裝elk 
    
                            docker pull sebp/elk
  5. 啟動elk 
    
                            sudo docker run -p 5601:5601 -p 9200:9200 -p 5044:5044 -it --name elk sebp/elk
  6. Dockerfile 執行 

    
                            docker build -t getting-started .
  7. 建立docker python 環境 
    
                            docker build -t isb_ugf_python38_image ./
  8. 進入該環境image 
    
                            docker-compose down && docker-compose up -d isb_ugf_python38 && docker-compose exec isb_ugf_python38 bash
  9. 安裝poety 
    
                            pip3 install poetry
  10. 進入app目錄下，利用poety安裝套件 
    
                            poetry install
    11. 　

poety教學 : https://myoceane.fr/index.php/python-poetry/
