Windows系統下遇到的坑

遇到更新window後會出現打不開docker的情形，錯誤代碼如下


                    error during connect: This error may indicate that the docker daemon is not running.:
                    Get "http://%2F%2F.%2Fpipe%2Fdocker_engine/v1.24/containers/json": open //./pipe/docker_engine: The system cannot find the file specified.

此時要下載 WSL2 Linux 核心更新套件 (適用於 x64 電腦)

參考 https://docs.microsoft.com/zh-tw/windows/wsl/install-win10#step-4---download-the-linux-kernel-update-package

如果還是不行，在電腦搜尋service打開服務

打開docker desktop service 讓狀態為執行中
