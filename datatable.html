DataTable 筆記

colvis 調整顯示欄位

文件路徑: https://datatables.net/reference/button/colvis


                    var table = $('#tradeTable2').DataTable({
                        dom: 'Brt',
                        paging: false,
                        ordering: false,
                        orderCellsTop: true,
                        columnDefs: [
                            {
                                targets: [ 2,4,6,7,9,14 ],
                                visible: false
                            }
                        ],
                        buttons: [
                            {
                                extend: 'colvis',
                                columns: ':not(.noVis)',
                                text: 'Select columns',
                                columnText: function ( dt, idx, title ) {
                                    return (idx + 1) + ': ' + title;
                                },
                            }
                        ]
                    });

columnDefs : 預設隱藏的欄位，targets 表示一開始顯示欄位 buttons : extend 引入 colvis，text 自訂欄位文字，columText 為點了button後個欄位文字

  • https://cdn.datatables.net/buttons/2.0.0/css/buttons.dataTables.min.css
  • https://cdn.datatables.net/1.11.1/js/jquery.dataTables.min.js
  • https://cdn.datatables.net/buttons/2.0.0/js/dataTables.buttons.min.js
  • https://cdn.datatables.net/buttons/2.0.0/js/buttons.colVis.min.js

文件路徑 : https://datatables.net/extensions/buttons/examples/html5/simple.html 


                    buttons: [
                       'excelHtml5',
                       {
                          extend: 'colvis',
                          columns: ':not(.noVis)',
                          text: 'Select columns'
                       }
